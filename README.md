# 🎂 Birthday Diary

A beautifully designed diary entry website dedicated to celebrating someone special. This is an elegant, modern web experience with smooth animations, custom interactions, and a dark aesthetic.

## 🌐 Live Demo

Visit the live site here: **[https://shaan-alpha.github.io/birthday-diary/](https://shaan-alpha.github.io/birthday-diary/)**

## ✨ Features

- **Custom Cursor**: Interactive cursor with smooth tracking and hover effects
- **Glassmorphic Design**: Modern frosted glass navigation bar with blur effects
- **Responsive Layout**: Optimized for desktop, tablet, and mobile devices
- **Modern Typography**: Beautiful fonts from Google Fonts (Fraunces, Instrument Serif, Geist)
- **Smooth Animations**: Elegant transitions and interactive elements
- **Dark Theme**: Easy on the eyes with a sophisticated dark background
- **Gradient Accents**: Vibrant color gradients (pink, orange, yellow) for visual appeal

## 🎨 Design Highlights

- **Navigation**: Fixed, centered glassmorphic navbar with smooth hover animations
- **Color Palette**: Dark background (#0a0a0f) with white text and gradient accents
- **Interactive Elements**: Hover effects on navigation links with underline animations
- **Mobile Optimized**: Responsive design with proper scaling and touch-friendly interactions

## 🛠️ Built With

- HTML5
- CSS3 (with advanced features like backdrop-filter and mix-blend-mode)
- Vanilla JavaScript (for custom cursor and interactions)
- Google Fonts
- GitHub Pages

## 🚀 Getting Started

### View Online
Simply visit the [live site](https://shaan-alpha.github.io/birthday-diary/) to experience the diary.

### Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/shaan-alpha/birthday-diary.git
   cd birthday-diary
   ```

2. Open `index.html` in your web browser:
   ```bash
   # On Windows
   start index.html
   
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   ```

Or use a local server (recommended):
```bash
python -m http.server 8000
# Then visit http://localhost:8000
```

## 📝 Customization

To customize this diary:

1. Edit the text in `index.html` to add your personal message
2. Modify the color gradients in the CSS (look for `linear-gradient` values)
3. Replace `music.mpeg` with your own audio file
4. Adjust fonts and styling as desired

## 🎵 Audio

The project includes a background music file (`music.mpeg`) that enhances the experience. You can replace it with your own audio file.
